# UI Panel Orchestrator

Panel frontend para orquestar arquitecturas, tasks, runs e inspección desde una sola interfaz.

## Stack

- Vite
- React 19
- TypeScript
- React Router
- Zustand
- TanStack Query

## Ejecutar

```bash
npm install
npm run dev
```

## Qué verás

- Sidebar con navegación principal
- Topbar con selector de arquitectura
- Panel central con dashboard, overview por arquitectura, tasks, runs y settings
- Inspector lateral para contexto rápido
- Bottom tray para logs, queue, events, locks y alerts

## Cómo usarlo

1. Abre el dashboard y elige una arquitectura desde la topbar.
2. Entra en Architectures y abre el overview de una arquitectura concreta.
3. Usa Tasks para revisar y filtrar trabajo del contexto activo.
4. Cambia a Runs para mirar ejecuciones del contexto activo o el historial completo.
5. Ajusta el shell desde Settings y prueba cómo persiste el estado al recargar.
